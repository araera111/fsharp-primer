コメントは、コードの説明を書くために使われます。コメントはコンパイルされません。

## コメントの書き方

コメントは、`//` で始まります。JavaScriptと同じです。

```fsharp
// これはコメントです
```

複数行のコメントは、`(*` と `*)` で囲みます。

```fsharp
(*
これは複数行のコメントです
*)
```

JavaScriptの場合は、こうです。

```javascript
/*
これは複数行のコメントです
*/
```

## まとめ

* 一行コメントは、`//` で始まります。
* 複数行のコメントは、`(*` と `*)` で囲みます。
